# 🎙️ Local Voice Assistant (Голосовой ассистент на Python)

Локальный голосовой ассистент с возможностью распознавания и синтеза речи, работающий **полностью офлайн** для базовых функций (благодаря Vosk) и использующий внешние легковесные API для получения актуальной информации.

Ассистент умеет проверять погоду в Санкт-Петербурге, давать рекомендации по поводу прогулок, а также записывать текстовые заметки под диктовку.

---

## ✨ Основные возможности

* **🎙️ Офлайн-распознавание речи:** Использует легковесную и быструю модель **Vosk** для распознавания команд на русском языке без отправки аудио на сторонние сервера.
* **🗣️ Синтез речи (TTS):** Отвечает пользователю голосом с помощью библиотеки `pyttsx3`, автоматически подстраиваясь под установленный в системе русский голосовой движок.
* **🌤️ Погодный модуль:** Интеграция с сервисом `wttr.in` для получения текущей погоды, скорости и направления ветра, а также умный анализ — стоит ли идти гулять.
* **📝 Голосовые заметки:** По команде ассистент переходит в режим диктовки, внимательно слушает текст и сохраняет его в файл `notes.txt` с фиксацией точного времени.

---

## 🛠️ Стек технологий и зависимости

* **Язык:** Python 3.x
* **Библиотеки:**
  * `vosk` — локальное распознавание речи.
  * `pyaudio` — захват аудиопотока с микрофона в реальном времени.
  * `pyttsx3` — текстовое воспроизведение (Text-to-Speech).
  * `requests` — HTTP-запросы к погодному API.
